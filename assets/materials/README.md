# Материалы сайта

Сюда складывать исходные и подключаемые материалы проекта:

- `logos/` — логотипы;
- `photos/` — фото и изображения;
- `videos/` — видео;
- `documents/` — документы;
- `other/` — прочие материалы.

После загрузки файлов в GitHub Pages ссылки будут строиться от корня репозитория, например:

`https://USERNAME.github.io/REPO/assets/materials/logos/logo.svg`
