# MEDIAMESTO / GitHub files for Tilda

Tilda остаётся основной страницей. GitHub хранит только кодовые файлы.

## Структура

```
assets/css/site.css
assets/js/site.js
pages/main/body.html
pages/blog/.gitkeep
pages/cases/.gitkeep
index.html
README.md
```

Папка с материалами не добавляется: логотипы, фото и видео обновляются вручную.

## Подключение в Tilda

CSS в HEAD:

```html
<link rel="stylesheet" href="https://stebunov22.github.io/mediamesto/assets/css/site.css?v=20260625-6">
```

JS перед `</body>`:

```html
<script src="https://stebunov22.github.io/mediamesto/assets/js/site.js?v=20260625-6"></script>
```
